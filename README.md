# LAN-Party-FAQ-Project
# 🎮 LAN Party FAQ Project

## 1. Justificació (Anàlisi d'UX)
La creació d'aquest document de FAQ's és vital per a l'èxit de la LAN Party. Des de la perspectiva de l'**Experiència d'Usuari (UX)**, una bona base de coneixement redueix l'estrès dels participants i optimitza el temps del personal tècnic. 

Hem analitzat els punts crítics (connexió, energia i programari) per garantir que qualsevol usuari, independentment del seu nivell tècnic, tingui una experiència fluida i sense interrupcions. Un usuari informat és un usuari que gaudeix de l'esdeveniment.

---

## 2. Document de FAQ's (Text Professional)
*Podeu trobar les preguntes detallades a continuació organitzades per categories:*

### 🌐 Xarxa i Connectivitat
* **Quina velocitat de xarxa tindrem?** Tindrem 1Gbps simètric per a tots els participants.
* **Cal portar cable de xarxa?** Sí, és obligatori portar un cable CAT6 de 5 metres.

### ⚡ Energia i Hardware
* **Quants endolls tindré disponibles?** Cada participant té accés a una toma de corrent. Es recomana portar una regleta.
* **Quins són els requisits mínims del PC?** Qualsevol PC capaç d'executar els jocs del torneig. Recomanem 16GB de RAM.
